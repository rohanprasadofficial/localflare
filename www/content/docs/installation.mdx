You can install Localflare globally or use npx to run it directly.

## Global Installation

```bash title="Terminal"
# Using npm
npm install -g localflare

# Using pnpm
pnpm add -g localflare

# Using yarn
yarn global add localflare
```

## Using npx (No Installation)

```bash title="Terminal"
npx localflare
```

<Callout type="info">
Using `npx` is recommended for one-off usage or to always use the latest version.
</Callout>

## Requirements

| Requirement | Version |
| --- | --- |
| Node.js | 18.0.0 or higher |
| Wrangler | Latest version (installed automatically) |
