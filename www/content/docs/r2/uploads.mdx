---
title: Uploads & Downloads
description: Upload and download files from R2 buckets.
---

## Uploading Files

- **Drag and drop** files directly into the file manager
- **Click Upload** to open the file picker
- **Multiple files** can be uploaded at once
- **Create folders** by specifying a prefix path before uploading

## Downloading Files

Click any object to open its detail view, then click the download button to save it locally. The original file name and content type are preserved.

## Bulk Operations

- **Download as ZIP** -- select multiple objects and download them as a single archive
- **Delete multiple** -- select objects and delete them in one action
- **Move to folder** -- move selected objects to a different prefix
