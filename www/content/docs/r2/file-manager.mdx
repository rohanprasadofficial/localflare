The file manager gives you a familiar file-system experience for browsing and managing objects in your R2 buckets.

## Navigation

- **Click folders** to drill into prefixes
- **Breadcrumbs** at the top of the panel let you jump back to any parent level
- **Search** to filter objects by key name
- **Sort** by name, size, or last modified date

## File Preview

Localflare can preview many file types directly in the dashboard:

- **Images** -- visual preview rendered inline (PNG, JPG, GIF, SVG, WebP)
- **Text / JSON / Code** -- syntax-highlighted preview with line numbers
- **Other** -- download option for unsupported formats
