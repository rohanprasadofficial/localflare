The table browser provides a visual overview of every table in your D1 database, including structure, row counts, and data.

## Table List

- View all tables in the connected D1 database
- See row counts at a glance
- Click a table to browse its rows
- Sort and filter data within any table

## Column Information

Each table displays detailed column metadata:

| Property | Description |
| --- | --- |
| Name | Column name |
| Type | SQLite data type (TEXT, INTEGER, REAL, BLOB) |
| Nullable | Whether the column allows NULL values |
| Default | Default value if one is set |
| Primary Key | Whether the column is part of the primary key |
