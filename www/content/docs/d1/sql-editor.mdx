---
title: SQL Editor
description: Powerful SQL editor for D1 databases.
---

The SQL editor lets you write and execute queries against your local D1 databases with a rich editing experience.

## Features

- Syntax highlighting for SQL
- Auto-completion for table and column names
- Query history across sessions
- Multiple result tabs for concurrent queries
- Export results as CSV or JSON

## Keyboard Shortcuts

| Shortcut | Action |
| --- | --- |
| Cmd/Ctrl + Enter | Execute query |
| Cmd/Ctrl + S | Save query |
| Cmd/Ctrl + / | Toggle comment |

## Example Queries

```sql
-- List all tables
SELECT name FROM sqlite_master WHERE type = 'table';

-- Query with pagination
SELECT * FROM users ORDER BY created_at DESC LIMIT 20 OFFSET 0;

-- Aggregate data
SELECT status, COUNT(*) as count
FROM orders
GROUP BY status;
```
