---
title: Instance Listing
description: View all active Durable Object instances.
---

The instance listing shows all active Durable Object instances grouped by class. Use it to find specific instances and navigate to their stored state.

## Instance Information

| Property | Description |
| --- | --- |
| ID | Unique identifier (hex string) |
| Name | Optional name if created with `idFromName()` |
| Class | The DO class name |
| Status | Active or hibernating |
