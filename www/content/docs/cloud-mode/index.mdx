Localflare isn't just for local development. With Cloud Mode, connect directly to your production Cloudflare account and browse live D1 databases, KV namespaces, R2 buckets, and Queues — all from the same dashboard you use for local development.

<Callout type="info">
Cloud Mode connects to the Cloudflare REST API using your API token. No wrangler process is required.
</Callout>

- **D1 Databases** — Query and browse your production databases
- **KV Namespaces** — Read and write keys in live namespaces
- **R2 Buckets** — Upload, download, and manage objects
- **Queues** — View queue configuration and status
